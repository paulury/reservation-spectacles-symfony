# Application de réservation de spectacles

Projet universitaire visant à concevoir une application web complète permettant la réservation de spectacles.

## Objectif du projet
Développer un site web dynamique permettant aux utilisateurs de consulter des spectacles, créer un compte et effectuer des réservations.

## Technologies utilisées
- PHP
- Symfony
- Composer
- EasyAdmin
- SQL / Base de données relationnelle
- Bootstrap (Front-end)

## Fonctionnalités principales

### Front-end
- Affichage des spectacles disponibles avec prix et places restantes
- Création de compte utilisateur
- Authentification utilisateur
- Réservation de spectacles
- Simulation de paiement avec récapitulatif de réservation

### Back-end
- Gestion CRUD :
  - Utilisateurs
  - Spectacles
  - Réservations
- Interface d’administration avec EasyAdmin
- Page de statistiques :
  - Nombre de places réservées
  - Revenus par spectacle

## Architecture
Le projet repose sur une architecture MVC fournie par Symfony avec gestion des entités et relations entre données.
